# Java值传递

**讲一下实参和形参的区别？**

答：

方法在调用和创建的时候都可能需要用到参数，这里分为两类：

- 实参：调用方法时要给方法传参数，这个参数必须是确定的值，这个参数就是实参。
- 形参：在定义方法时要定义一些参数接收实参，参数可以不是确定的值，这个参数就是形参。
